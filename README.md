# Daily Check-In

A guided daily check-in webapp that helps you start your day with intention. The app walks you through a short sequence of mindfulness steps:

1. **Breathe**: A guided breathing exercise to calm and centre yourself.
2. **Feel**: Reflect on how you're feeling right now, without judgement.
3. **Gratitude**: Name something you're grateful for today.
4. **Intention**: Set a positive intention to carry into the day.

## Getting Started

```
pnpm install
pnpm start
```

`pnpm start` brings up the full application (frontend and any supporting backend services).
